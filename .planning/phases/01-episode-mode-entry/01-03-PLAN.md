| Boundary | Description |
|----------|-------------|
| selected episode query -> script-page selector UI | The visible mode could drift from persisted state if the UI creates a local mirror |
| selector/CTA interaction -> launch path | The CTA could still launch an implicit default if the mode is only displayed, not actually read at action time |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03-01 | T | `ScriptViewAssetsPanel.tsx` | mitigate | Render selector state directly from `episodeProductionMode` and update through the runtime callback; do not add standalone `useState` as a second source of truth. |
| T-01-03-02 | I | CTA helper text + launch handler | mitigate | Keep the CTA base label intact but add explicit path copy and branch the action from `episodeProductionMode` so the UI and behavior cannot disagree. |
