# Phase 01 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Phase Contract

- Placement is fixed: render the episode mode selector in the right-side script-page action stack, directly below the `剧中资产` card and directly above the `确认并开始绘制` button.
- Structure is fixed: use one compact selector card with two mutually exclusive options, `多镜头片段模式` and `传统模式`.
- Default is fixed: newly configured episodes enter with `多镜头片段模式` preselected.
- Feedback is fixed: the selector card and CTA area must always show the active production path before launch.
- Switching is fixed: switching is immediate for episodes with no downstream artifacts; switching a started episode opens the existing warning-style confirmation dialog pattern.
- Design-system assumption: keep the existing repo-local glass design system for this phase. `components.json` is absent, and the workflow fallback is `Tool: none` rather than introducing shadcn mid-phase.

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Badge-dot offsets, inline icon gaps |
| sm | 8px | Gap between mode icon and label, helper-copy spacing |
| md | 16px | Default card padding, vertical spacing inside the selector card |
| lg | 24px | Gap between asset card, mode selector card, and CTA |
| xl | 32px | Right-panel section separation on desktop |
| 2xl | 48px | Major card-group break if an additional warning block appears |
| 3xl | 64px | Page-level right-column breathing room only |

Exceptions: 44px minimum hit area for icon-only or segmented-toggle targets if any compact toggle affordance is introduced.

## Color

|------|-------|-------|
| Dominant (60%) | `#f3f4f6` | Workspace canvas and surrounding page background |
| Secondary (30%) | `rgba(255, 255, 255, 0.88)` | Selector card, helper card, modal surface, and supporting panel surfaces |
| Accent (10%) | `#2f7bff` | Selected mode state, primary CTA fill, focus ring, active path badge |
