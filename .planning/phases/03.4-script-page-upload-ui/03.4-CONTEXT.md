# Phase 03.4 Context — Script Page Upload UI

## Goal

Add upload storyboard package UI to the script page multi-shot mode so users can import director-authored package files instead of confirming auto-generated drafts.

## Scope

- Add “上传分镜表” button beside “确认并开始绘制”.
- Read `.md` / `.json` files from browser.
- Call import preview API.
- Show preview: segments, shot counts, asset matching, overwrite behavior.
- Confirm commit and navigate to multi-shot confirmation / video stage.
- Add focused UI tests.

## Non-Goals

- No package parser/schema work; that belongs to Phase 03.2.
- No import API persistence work; that belongs to Phase 03.3.
- Do not trigger image/video generation automatically after import.
- Do not bypass user confirmation for overwrite behavior.

## Depends On

- Phase 03.2 — Storyboard Package Import Contract.
- Phase 03.3 — Import API And Persistence.

## Requirements

- SBPI-22 through SBPI-30 in `.planning/REQUIREMENTS.md`.
