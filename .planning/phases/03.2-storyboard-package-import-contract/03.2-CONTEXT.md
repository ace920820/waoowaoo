# Phase 03.2 Context — Storyboard Package Import Contract

## Goal

Define the `waoo.storyboard_package` v1.0 contract for importing director-authored storyboard packages. Implement parser, validator, mapper unit tests, and Markdown fenced JSON block extraction.

## User Request

External director-agent teams produce higher-quality storyboard packages from scripts. Their package includes storyboard script, production shot table, and image-generation prompts. Waoo should support importing a stable machine-readable package format so later phases can prefill multi-shot confirmation and video production fields.

## Scope

- Define schema for `waoo.storyboard_package` v1.0.
- Support `.json` files and Markdown fenced blocks using `waoo-storyboard-package+json`.
- Validate scenes, segments, `templateKey`, storyboard mode, video config, assets, `cinematicPlan`, and shots.
- Map imported package into internal multi-shot draft / shot-group persistence payloads.
- Add unit tests for parser, validator, and mapper.

## Non-Goals

- No import API persistence yet; that belongs to proposed Phase 03.3.
- No script-page upload UI yet; that belongs to proposed Phase 03.4.
- Do not freely guess arbitrary Markdown tables. Markdown import should require the explicit fenced JSON block.
- Do not create project assets silently during mapping.
